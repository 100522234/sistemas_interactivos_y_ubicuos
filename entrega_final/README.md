### Instrucciones de Ejecución:
1. Descomprimir el archivo y entrar en la carpeta `src`
2. Abrir una terminal en dicha carpeta
3. Ejecutar `npm install` para instalar las dependencias (Socket.IO, Express, etc.)
4. Conectarse a la base de datos MongoDB
5. Ejecutar el servidor con `node server.js`
6. en otra terminal de visual studio poneis ipconfig y copiais la direccion IPV4
7. Conectar el dispositivo móvil usando `chrome://inspect/#devices`
8. En el movil pone http://X.X.X.X:3000/index.html  (poner la IP en X.X.X.X) 
5. Desde el movil abrir `chrome://flags/`
6. Ir a "insecure origins treated as secure" y poner http://X.X.X.X:3000
7. Reiniciar el buscador de google y listo!
